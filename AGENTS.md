# 《炉心残冬》仓库协作说明

## 项目范围

本仓库用于《炉心残冬》的文档、数据、源文件与测试资料管理。

## 协作规则

- 新增或修改内容前先阅读 `README.md`、`UPLOAD_GUIDE.md` 与 `docs/INDEX.md`。
- 未经明确授权，不删除或覆盖已有文件。
- 文档按用途放入 `docs/` 对应目录；补丁资料按编号归档。
- 游戏代码放入 `src/`，测试放入 `tests/`；当前已完成至代码 Patch 006 的 006B 科技研究、已确认科技效果与过载压力闭环，006C、事件、承诺和终局必须按后续 Patch 分批实现。
- 废弃资料移入 `docs/deprecated/`，并说明废弃原因与替代位置。
