## 一、合并后的干净版正文  
**1. 定位**  
本节处理炉城居民的娱乐、安抚与精神减压方式。  
它解决的问题是：  
```
人还活着，但精神已经快撑不住了。

```
本节包括：  
* 火盆集会。  
* 争端裁决。  
* 小酒馆。  
* 大赌场。  
* 温泉汤浴。  
* 娱乐线对恐慌、信任、疲劳、疾病风险与终局评价的影响方向。  
本节不处理：  
* 精神 / 秩序路线。  
* 宗教或管控建筑。  
* 社会情绪每日上限。  
* 具体信任 / 恐慌数值。  
* 资源消耗最终平衡表。  
* 固定事件完整文本。  
说明：  
* 娱乐减压线只能缓解日常压力。  
* 娱乐减压线不能替代精神 / 秩序路线。  
* 娱乐减压线不能压住死亡、医疗崩溃、炉心熄灭、饥荒、第七霜落等重大社会危机。  
   
⸻  
   
**2. 分支结构**  
娱乐减压线无主线互斥。  
完整结构：  
```
火盆集会
├─ 争端裁决
└─ 小酒馆
   └─ 大赌场
      └─ 温泉汤浴

```
说明：  
* 火盆集会是基础公共安抚设施。  
* 争端裁决强化火盆集会。  
* 小酒馆解锁正式娱乐建筑。  
* 大赌场解锁正式高级娱乐建筑。  
* 温泉汤浴是大赌场附属功能，不是独立建筑。  
   
⸻  
   
**3. 娱乐线强度边界**  
娱乐减压线的定位是：  
```
前期提供基础恐慌缓冲。
中后期提供低水平维持。
不能替代精神 / 秩序路线。
不能达到或超过精神 / 秩序类建筑与行动的降压强度。

```
规则：  
```
娱乐减压来源的恐慌降低效果，整体强度必须低于精神 / 秩序路线。
娱乐建筑只能缓解日常压力，不能压住重大社会危机。
娱乐线不能抵消死亡、医疗崩溃、炉心熄灭、饥荒、第七霜落等重大事件。

```
设计定位：  
```
火盆集会、小酒馆、大赌场、温泉汤浴：
提供前期安抚与后期低水平维持。

精神 / 秩序路线：
提供中后期更强的社会控制、信念支撑或危机压制能力。

```
说明：  
娱乐线让人暂时喘气。 精神 / 秩序线才是城市后期面对崩溃时的深层社会结构。  
娱乐可以让人忘记今晚很冷，不能让人相信冻死也有意义，更不能让人害怕到不敢崩溃。  
   
⸻  
   
**4. 火盆集会**  
前置：无 状态：开局可选炉律之一 类型：公共集会点 解锁类型：公共设施 是否正式建筑：否 是否占槽：否 数量限制：全城最多 1 个  
规则：  
```
火盆集会不作为正式建筑。
火盆集会不占普通建筑槽位。
火盆集会不需要 build 命令。
火盆集会视为路口、炉边或公共空地中的集会点。
签署后启用，全城最多 1 个。
炉心运行时效果正常。
炉心关闭或燃料严重不足时，效果暂停。

```
效果方向：  
```
每日小幅降低恐慌。
不直接解决食物、燃料、医疗或住房问题。
不压制重大社会危机。

```
签署瞬间影响方向：  
```
信任：无变化
恐慌：微微下降

```
文案方向：  
```
准备签署：火盆集会

炉城将在炉边与路口留出几处可以聚集的火盆。
人们会在那里讲些不那么冷的旧事，抱怨今天的风，也假装明天还值得撑一撑。

效果：
- 启用火盆集会。
- 火盆集会不是正式建筑。
- 火盆集会不占普通建筑槽位。
- 全城最多 1 个。
- 炉心正常运行时，每日小幅降低恐慌。

```
说明：  
火盆集会是最低层的公共安抚设施。 它不能无成本叠加，也不能在炉心关闭或燃料崩溃时继续提供完整安抚。  
   
⸻  
   
**5. 争端裁决**  
前置：火盆集会 类型：火盆集会强化炉律 解锁类型：火盆集会强化规则 是否解锁建筑：否 是否占槽：否 互斥：无  
定位：  
争端裁决把私下冲突转为公开处理。 它不是娱乐建筑，而是防止恐慌演变成斗殴、骚乱或失控事件的社会缓冲规则。  
公开裁决能减少私下积怨，但也可能让矛盾在火盆前爆开，造成轻微斗殴或受伤。  
  
效果方向：  
```
降低冲突类事件压力。

```
```
降低恐慌型社会失控风险。
可能带来超小幅轻伤 / 斗殴受伤风险。
不新增建筑槽位。

```
签署瞬间影响方向：  
```
信任：微微下降
恐慌：微微下降

```
文案方向：  
```
准备签署：争端裁决

争执会被带到火盆前。
有人说理，有人认错，也有人只是学会了在更亮的地方闭嘴。

但不是每场争端都会安静收尾。
有些话说出口以后，拳头也会跟着落下。

效果：
- 强化火盆集会。
- 降低冲突类事件压力。
- 降低恐慌型社会失控风险。
- 可能带来轻伤/斗殴受伤风险。

```
说明：  
争端裁决不是管控路线，也不是精神 / 秩序路线。 它只是在火盆集会基础上提供更稳定的低强度社会缓冲。  
   
⸻

大赌场后方会引出一处被炉热勉强维持的汤浴。
水并不总是清澈，热气却足够让人短暂忘记外面的风。

**14. 本节结论**  
娱乐减压线形成递进结构：  
```
火盆集会：
基础公共减压，不占槽，最多 1 个。

争端裁决：
强化火盆集会，降低冲突类事件压力。

小酒馆：
正式娱乐建筑，占 1 槽，最多 1 座，需要人手与温度，每日降低恐慌。

大赌场：
高级娱乐建筑，最多 1 座，占 3 槽，强力降低恐慌，签署时提高信任，并进入终局标签 / 沉溺性安抚记录。

温泉汤浴：
大赌场附属功能，不占槽，进一步安抚居民，但带来超小幅疾病传播风险。

```
最终分支树：  
```
火盆集会
├─ 争端裁决
└─ 小酒馆
   └─ 大赌场
      └─ 温泉汤浴

```
核心原则：  
娱乐减压线让人暂时喘气，但不能替代城市的深层社会结构。 它能让居民在炉火旁笑一会儿，在酒馆里忘一会儿，在赌桌前麻木一会儿。 但它不能让死者复活，不能让病床变多，不能让炉心重新点燃，也不能让第七霜落变得仁慈。
