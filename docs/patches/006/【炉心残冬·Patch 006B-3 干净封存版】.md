
**4. 地图资源结构**  
**4.1 地表小型物资点**  
开局地图会生成若干地表小型物资点。  
规则：  
```
地表小型物资点开局固定生成。
无人采集时不会减少。
不会自动刷新。
不会凭空变多。
采完后进入枯竭状态。

```
类型包括：  
```
地表木材
地表煤炭
地表钢铁

```
V1 不包含：  
```
散落食物点
地表食物
可采集食物残余点

```
说明：  
* 地表小型物资点是早期缓冲资源。  
* 它们不是长期资源系统。  
* “其他可拾取残余资源”旧表述容易误导为散落食物点，本版删除。  
* 若后续需要新增其他可拾取资源，必须单独定义资源类型，不得默认并入食物。  
   
⸻  
   
**4.2 伐木场点**  
地图固定存在 2 个伐木场点。  
规则：  
```
每局固定 2 个伐木场点。
伐木场点不随机数量。
玩家需要研究木材加工 I 后，才能建造正式伐木场。
伐木场只能建在伐木场点上。

```
说明：  
前期依靠地表木材过渡。 研究木材加工 I 后，玩家才能建立稳定木材来源。  
   
⸻  
   
**4.3 小型采煤机 / 小型采钢机**  
小型采煤机和小型采钢机不绑定大型矿点。  
规则：  
```
小型采煤机不需要大型煤矿点。
小型采钢机不需要大型钢铁矿点。
它们属于可建造的中期资源生产建筑。

```
限制：  
```
需要科技解锁。
需要建造材料。
占普通建筑槽位。
需要人手。
产出低于大型矿点机器。
受温度、疾病、人手、停工规则影响。
有全城上限或软上限，具体数值后续平衡。

```
说明：  
* 小型采煤机 / 小型采钢机是城市内浅层资源开采装置。  
* 它们不是矿点建筑。  
* 它们不是“小型地下煤铁点”。  
* 它们不是无限铺开的地下印钞机。  
   
⸻  
   
**4.4 大型矿点与大型矿机**  
大型煤矿机和大型钢铁矿机必须绑定地图矿点。  
规则：  
```
大型煤矿机只能建在大型煤矿点上。
大型钢铁矿机只能建在大型钢铁矿点上。
没有对应矿点，不能建造对应大型矿机。

```
说明：  
* 大型矿点是地图资源结构的一部分。  
* 大型矿机不是随意安放建筑。  
* 大型矿点本身不占普通建筑槽位。  
* 绑定矿点的大型矿机占普通建筑槽位。  
   
⸻  
   
### 5. 地图资源模板暂定口径  
  
006B-3 只记录资源科技与资源点使用规则，不最终锁定地图模板平衡。  
  
本节只确认：  
  
- 地图中可以存在大型煤矿点。  
- 地图中可以存在大型钢铁矿点。  
- 地图中可以存在伐木场点。  
- 大型煤矿机必须绑定大型煤矿点。  
- 大型钢铁矿机必须绑定大型钢铁矿点。  
- 伐木场必须绑定伐木场点。  
  
以下内容不在 006B-3 中封存：  
  
- 各地图模板的大型煤矿点数量。  
- 各地图模板的大型钢铁矿点数量。  
- 各地图模板的伐木场点数量。  
- 三张地图资源总量是否相同。  
- 地图是否存在补偿。  
- 地图资源距离、风险、储量补偿。  
- 地图模板名称与最终结构。  
  
说明：  
  
006B-3 只处理资源科技。  
地图模板、资源总量、资源补偿与地图名，交由后续 008 地图系统统一收口。

**7. 科技树总览**  
006B-3 分为五条线：  
```
A. 地表采集线
B. 木材 / 伐木场线
C. 小型煤炭 → 大型煤矿线
D. 小型钢铁 → 大型钢铁矿线
E. 仓储线

```
科技树关系：  
```
散落采集工具
  └─ 避寒采集棚改良

木材加工 I
  └─ 木材加工 II

煤层支架
  └─ 小型采煤改良
        └─ 深层煤脉开采

钢材筛选
  └─ 小型采钢改良
        └─ 深层钢脉开采

深井矿架
  ├─ 深层煤脉开采
  └─ 深层钢脉开采

仓储扩容

```
说明：  
* 深井矿架是大型矿机的通用工程前置。  
* 深层煤脉开采解锁大型煤矿机。  
* 深层钢脉开采解锁大型钢铁矿机。  
* 仓储线独立，不作为煤线或钢铁线硬前置。
