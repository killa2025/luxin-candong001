# A 【炉心残冬·Patch 006A-5 干净封存版】  
主题：社会适配城约 A5 —— 医疗处置线 状态：干净封存 归属：006A 社会适配城约  
   
⸻  
   
## 一、合并后的干净版正文  
**1. 定位**  
本节处理炉城的医疗处置规则。  
它解决的问题是：  
```
病人太多、床位太少、资源不够时，
炉城如何决定治疗方式、收治容量与救治优先级。

```
本节包括：  
* 基础医疗法。  
* 临时救护。  
* 稳定疗法。  
* 激进疗法。  
* 扩容收治。  
* 额外医疗配给。  
* 分级救治。  
* 养护所。  
* 医疗线防死局规则。  
本节不处理：  
* 医院科技。  
* 义肢生产。  
* 药品生产科技。  
* 医疗建筑升级科技。  
* 社会情绪每日上限。  
* 具体信任 / 恐慌数值表。  
* 具体治疗效率数值表。  
说明：  
* 义肢属于后续工程科技，不进入 006A 医疗处置线。  
* 医院不由 006A 直接解锁。  
* 基础医疗法只解锁正式医疗站，不自动赠送医疗站建筑。  
   
⸻  
   
**2. 分支结构**  
医疗处置线为部分互斥结构。  
```
基础医疗法
├─ 稳定疗法 ↔ 激进疗法
├─ 扩容收治
│  └─ 养护所
├─ 额外医疗配给
└─ 分级救治

```
其中：  
```
稳定疗法 ↔ 激进疗法

```
二者互斥。  
说明：  
* 基础医疗法是医疗线入口。  
* 稳定疗法与激进疗法只能选择其一。  
* 扩容收治不是新建筑，是医疗容量规则变化。  
* 额外医疗配给不是每日被动扣粮，是主动医疗行动。  
* 分级救治不是每日被动制度，是医疗超载时的主动危机行动。  
* 养护所是独立建筑，用于安置伤残人口。  
   
⸻  
   
**3. 默认临时救护**  
为避免开局医疗死局，游戏默认存在极低容量的临时救护能力。  
规则：  
```
未签署基础医疗法前，城市仍可进行临时救护。
临时救护容量极低，只能处理少量轻症。
临时救护不是建筑。
临时救护不占槽。
临时救护不能替代正式医疗站。
签署基础医疗法后，玩家才可建造正式医疗站。

```
设计目的：  
* 避免玩家因开局未签基础医疗法而立刻医疗死锁。  
* 保留基础医疗法解锁正式医疗站的战略价值。  
* 让医疗压力从开局就可见，但不能让临时救护变成免费医疗站。  
   
⸻  
   
**4. 基础医疗法**  
前置：无 状态：开局可选炉律之一 类型：医疗线基础炉律 解锁类型：正式建筑 + 医疗系统入口 解锁建筑：医疗站 是否占槽：是  
医疗站建筑规则：  
```
医疗站：小型建筑
占 1 槽
推荐区域：内环 / 中环
最多 4 座
需要人手运行
需要较高温度
低温不足时停治
可应急加热

```
基础医疗法签署后：  
```
解锁医疗站建筑。
解锁后续医疗处置炉律。
允许正式收治轻症与重症。
医疗面板开始明确显示轻症、重症、伤残、医疗容量与医疗压力。

```
签署瞬间影响方向：  
```
信任：微微上升
恐慌：无变化

```
文案方向：  
```
准备签署：基础医疗法

医护人员将不再只按哭喊声决定先救谁。
炉城会建立正式医疗站，把病人分为轻症、重症与无法立刻治愈的人。

这听起来冷酷，但混乱本身也会死人。

效果：
- 解锁医疗站建筑。
- 解锁后续医疗处置炉律。
- 医疗面板显示病患分类与医疗压力。

```
说明：  
* 医疗站不是开局默认可建建筑。  
* 开局只有临时救护。  
* 医疗站必须通过基础医疗法解锁后建造。  
* 医院最多 2 座，但医院不由 006A 直接解锁。  
   
⸻  
   
**5. 稳定疗法**  
前置：基础医疗法 互斥：激进疗法 解锁类型：医疗结算规则 是否解锁建筑：否  
定位：  
稳定疗法偏向保守救治。 它降低重症死亡压力，但治疗更慢，占床更久。  
效果方向：  
```
重症死亡压力下降
重症治疗速度下降或占床时间上升
治疗进度仍然每天推进
不会导致病人无限期占床

```
签署瞬间影响方向：  
```
信任：微微上升
恐慌：微微上升

```
文案方向：  
```
准备签署：稳定疗法

医生会先让濒死的人稳住呼吸。
他们可能没法立刻回到岗位，但至少不会在今晚被风雪和高烧一起带走。

效果：
- 重症死亡压力下降。
- 重症治疗更慢，或占床更久。
- 激进疗法路线永久锁定。

```
说明：  
* 稳定疗法不能让病人无限期占床。  
* 稳定疗法不是医疗容量扩容。  
* 稳定疗法降低死亡压力，但会拖慢床位周转。  
   
⸻  
   
**6. 激进疗法**  
前置：基础医疗法 互斥：稳定疗法 解锁类型：医疗结算规则 是否解锁建筑：否  
定位：  
激进疗法偏向快速处置。 它提高重症处理速度，但伤残风险上升。  
效果方向：  
```
重症治疗速度提高
部分重症更快脱离危险
伤残风险上升
伤残者仍消耗食物、住房，并造成长期劳动力压力

```
签署瞬间影响方向：  
```
信任：微微上升
恐慌：微微上升

```
文案方向：  
```
准备签署：激进疗法

医生将被允许采取更激进的处置。
在炉城，保住一条命有时意味着失去一部分身体。

这不是奇迹，只是用更少的时间换更重的代价。

效果：
- 重症处理速度提高。
- 伤残风险上升。
- 稳定疗法路线永久锁定。

```
说明：  
* 激进疗法不能无代价清空重症压力。  
* 它只是把部分死亡压力转化为伤残与长期负担。  
* 伤残者不会恢复普通劳动力身份，除非后续工程科技另行定义义肢或恢复系统。  
   
⸻  
   
**7. 扩容收治**  
前置：基础医疗法 互斥：无 解锁类型：医疗容量规则 是否解锁建筑：否  
定位：  
扩容收治用于缓解医疗容量不足。 它增加床位，但治疗质量下降。  
效果方向：  
```
医疗站 / 医院容量提高
治疗效率下降或恢复速度下降
医疗拥挤状态缓解
解锁后续养护所炉律

```
签署瞬间影响方向：  
```
信任：无变化
恐慌：微微上升

```
文案方向：  
```
准备签署：扩容收治

走廊会摆上床板，炉边会挤进更多病人。
他们终于不用躺在雪地里等死，但医生经过每一张床的时间也会变得更短。

效果：
- 医疗容量提高。
- 治疗质量下降，或恢复速度下降。
- 解锁后续养护所炉律。

```
说明：  
* 扩容收治不是新建筑。  
* 它是医疗站 / 医院的收治规则变化。  
* 它不能让医疗系统无代价扩容。  
* 它不直接恢复伤残者劳动能力。  
   
⸻

**9. 分级救治**  
前置：扩容收治 + 额外医疗配给 互斥：无 类型：医疗危机行动解锁炉律 解锁类型：医疗危机行动 是否解锁建筑：否 是否必须确认：是  
定位：  
分级救治是高收益、高代价的医疗危机按钮。 它不是全城医疗总开关，而是对指定医疗建筑执行的一次危机处置。  
签署“分级救治”只代表解锁该行动。 签署瞬间不直接改变信任 / 恐慌。  
真正的信任 / 恐慌波动，由启动分级救治后产生的死亡、伤残、放弃治疗等结果，交给死亡系统与社会情绪系统统一结算。  
  
准备签署：分级救治  
  
医疗站已经无法收治所有人。  
医生将被允许在指定医疗站或医院中，优先救治最可能活下来的人。  
  
一部分病人会立刻脱离危险。  
也会有一部分人被延后、被放弃导致伤残，或在粗暴处置中永远离开。  
  
它只会在执政官主动指定建筑并确认启动时生效。  
  
效果：  
  
* 解锁“分级救治 <建筑编号>”行动。  
* 只能指定医疗站或医院。  
* 不能指定养护所。  
* 可立即治愈或稳定一部分病人。  
* 可能造成一部分病人死亡或伤残。  
  
确认请输入：  
confirm sign 分级救治  
  
命令方向：  
```
分级救治 <建筑编号>

```

```

分级救治 医疗站#2
分级救治 医院#1

```
可指定建筑：  
```
医疗站
医院

```
不可指定建筑：  
```

大赌场
墓园
冷藏坑

住宅
其他非医疗治疗建筑

```
说明：  
* 养护所用于伤残人口安置，不是急救治疗建筑。  
* 分级救治不能指定养护所。  
* 分级救治不能覆盖全城医疗系统。  
* 每次启动只作用于一个指定医疗建筑。  
使用条件：  
```
目标建筑必须是医疗站或医院。
目标建筑必须正常运行。
目标建筑内存在可处置病人。
目标建筑处于医疗超载或危机压力状态。
分级救治行动不在冷却中。

```
不可用提示方向：  
```

```

```

```

```

被放弃或低优先级处理的病人可能死亡或转为伤残。
目标建筑医疗压力立刻下降。
```

**16. 本节结论**  
医疗处置线形成多层取舍：  
```
基础医疗法：
建立正式医疗体系，解锁医疗站。

稳定疗法：
少死，但慢，占床久。

激进疗法：
救得快，但伤残风险高。

扩容收治：
床位更多，但质量下降。

额外医疗配给：
花食物换治疗推进，温和稳定。

分级救治：
医疗爆仓时强行提高处理量，收益高，代价残酷。

养护所：
安置伤残，降低长期压力，但不恢复劳动力。

```
最终分支树：  
```
基础医疗法
├─ 稳定疗法
├─ 激进疗法
├─ 扩容收治
│  └─ 养护所
├─ 额外医疗配给
└─ 分级救治（前置：扩容收治 + 额外医疗配给）

```
其中：  
```
稳定疗法 ↔ 激进疗法

```
互斥。  
核心原则：  
医疗线不是让城市无痛消除疾病。 它只是在病人太多、床位太少、食物不够的时候，让执政官选择把代价放在哪里：死亡、伤残、食物、床位、信任、恐慌，或一条日后仍会被记住的伦理底线。
