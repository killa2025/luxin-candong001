# 【炉心残冬·Patch 006A-2 干净封存版】  
主题：社会适配城约 A2 —— 工时 / 劳作作息线 状态：干净封存 归属：006A 社会适配城约  
   
⸻  
   
## 一、合并后的干净版正文  
**1. 定位**  
本节处理成年人及普通工作建筑的工时压榨规则。  
它不改变儿童路线、不改变医疗制度、不改变精神 / 秩序路线，只处理：  
* 临时提高单建筑产出。  
* 长期提高全城普通工作建筑产出。  
* 因加班导致的患病、事故、恐慌与信任代价。  
* 加班日与长班制的命令、持续时间、风险提示和防误触规则。  
核心取舍：  
```
更高产出
换取
更高风险、更高恐慌、更低信任

```
   
⸻  
   
**2. 分支结构**  
工时 / 劳作作息线无互斥。  
```
加班日
└─ 长班制

```
两项炉律均属于社会适配城约。 签署后进入社会适配城约固定冷却。  
“应急班次”为旧名，正式名称统一为：  
```
加班日

```
   
⸻  
   
**3. 工时类型**  
V1 工时口径统一为：  
```
标准工时：8 小时
长班制：10 小时
加班日：24 小时

```
说明：  
* 标准工时是默认状态。  
* 长班制是全城持续工时制度。  
* 加班日是单建筑 24 小时极限急令。  
* 加班日不是普通延长班，也不是可随时取消的轻度加班。  
   
⸻  
   
**4. 加班日**  
前置：无 类型：单建筑当日急令 状态：开局可选炉律之一 是否需要确认：是  
签署后解锁单建筑操作：  
```
加班日 <建筑编号>

```
兼容命令：  
```
overtime <建筑编号>

```
执行需要确认：  
```
加班日 <建筑编号>
confirm 加班日 <建筑编号>

```
   
⸻  
   
**4.1 加班日规则**  
```
加班日只能指定一座正在运行且有人手的可加班工作建筑。
当日该建筑按 24 小时极限急令结算。
当日该建筑产出提高。
当日该建筑患病 / 疲劳 / 事故风险提高。
每日最多使用 1 次。
日结后自动关闭。
第二天不持续。

```
加班日属于高风险操作。 原因：  
```
24 小时极限行动
不可主动取消
可能触发伤病、事故、死亡或固定事件

```
   
⸻  
   
**4.2 加班日可用目标**  
V1 加班日可作用于：  
```
医疗站
医院
研究所
食堂
温室
小型采煤机
小型采钢机
大型煤矿机
大型钢铁矿机
伐木场
其他建筑表明确标记为可加班日的普通生产 / 工作建筑

```
默认不可作用于：  
```
儿童保护所
学堂
小酒馆
大赌场
墓园
冷藏坑
仓库
住宅
火盆集会
其他非工作建筑 / 非生产建筑

```
说明：  
* 儿童保护所、学堂不属于普通生产建筑，默认不可加班日。  
* 小酒馆 / 大赌场属于娱乐建筑，不作为加班日压榨目标。  
* 修造工坊已从 V1 正式建筑中删除，因此不属于加班日目标。  
* 其他建筑若后续需要支持加班日，必须在建筑表中单独标记。  
   
⸻  
   
**4.3 加班日不可用情况**  
不可使用情况：  
```
目标建筑无人
目标建筑停工
目标建筑因温度不足停摆
目标建筑不属于可加班工作建筑
目标建筑不在可加班目标列表中
当日已使用过加班日

```
失败提示方向：  
```
加班日失败：该建筑无人工作或已经停工，无法继续压榨空气。

```
目标不合法提示方向：  
```
加班日失败：该建筑不属于可加班工作建筑。

```
   
⸻  
   
**4.4 加班日不可主动取消**  
加班日一旦启动，玩家不能主动取消。  
规则：  
```
不能通过 unassign 撤出工人来取消加班日。
不能通过切换工时制度提前结束加班日。
不能通过再次输入标准工时取消加班日。

```
若建筑因系统原因无法继续运行，则按异常中断处理。  
异常中断条件包括：  
```
低温停工
燃料崩溃
灾害强制停工
建筑被硬规则关闭
其他系统级停工

```
说明：  
* 加班日是当日急令，不是普通开关。  
* 玩家启动后，必须承担当日风险。  
* 系统级停工可以中断加班，但不代表玩家可主动撤销代价。  
   
⸻  
   
**4.5 首次加班死亡事件钩子**  
首次启用 24 小时加班后，可触发一次固定死亡事件。  
事件方向：  
```
首次 24 小时加班后，触发固定死亡事件。
该事件用于让玩家理解加班日的风险。

```
该事件可特殊允许玩家选择是否停止本次加班。  
说明：  
* 这是固定事件的特殊处理。  
* 不代表平时可以手动取消加班日。  
* 固定事件完整文本与触发条件留到固定事件系统处理。  
   
⸻  
   
**4.6 加班日签署文案方向**  
签署提示：  
```
准备签署：加班日

炉城将允许执政官在必要时指定一座工作建筑进行 24 小时极限劳作。
这不会让人变得更强，只会让他们在倒下前多搬一筐煤、多切一捆木、多熬一锅汤。

效果：
- 解锁“加班日 <建筑编号>”操作。
- 每日最多指定一座正在运行的可加班工作建筑。
- 该建筑当日产出提高。
- 该建筑当日患病、疲劳与事故风险提高。
- 加班日需要二次确认。
- 加班日一旦启动，通常不能主动取消。

确认请输入：
confirm sign 加班日

```
执行确认提示：  
```
准备启动加班日：<建筑编号>

这是 24 小时极限急令。
启动后，该建筑今日不会按时停工，且通常不能主动取消。
工人、儿童辅工或学徒若在该建筑内工作，将一并承受额外患病、疲劳、事故与死亡风险。

确认请输入：
confirm 加班日 <建筑编号>

```
执行成功提示：  
```
加班日已启动。

这座建筑今晚不会按时熄灯。
炉城会得到更多产出，代价则写在工人的骨头里。

```
   
⸻  
   
**5. 长班制**  
前置：加班日 类型：全城工作制度  
签署后解锁全城工时模式：  
```
工时 标准
工时 长班

```
兼容命令：  
```
worktime normal
worktime long

```
   
⸻  
   
**5.1 长班制规则**  
```
长班制可由玩家开启或关闭。
开启后持续生效，直到玩家手动关闭。
长班制影响所有普通工作建筑。
开启期间，普通工作建筑每日产出提高。
开启期间，普通工作人口患病 / 疲劳 / 事故风险提高。
开启期间，每日小幅提高恐慌。
开启期间，每日缓慢降低信任。

```
长班制不是每日按钮。 它是一个可开关的全城持续制度。  
   
⸻  
   
**5.2 长班制影响范围**  
长班制影响：  
```
普通生产建筑
普通加工建筑
普通资源建筑
普通工作建筑
建筑表明确标记为受长班制影响的建筑

```
长班制默认不影响：  
```
儿童保护所
学堂
小酒馆
大赌场
墓园
冷藏坑
仓库
住宅
火盆集会
其他非普通生产 / 工作建筑

```
说明：  
* 儿童保护所、学堂不吃长班制。  
* 小酒馆 / 大赌场不吃长班制。  
* 仓库、住宅、墓园、冷藏坑等非工作或非生产建筑不吃长班制。  
* 修造工坊已从 V1 正式建筑中删除，不进入长班制目标表。  
   
⸻  
   
**5.3 无人建筑规则**  
```
无人建筑自动停工。
无人建筑不产生收益，也不承受长班制风险。
长班制不会因为某座建筑无人而自动关闭。
若第二天重新分配人手，该建筑按当前工时制度结算。

```
   
⸻  
   
**5.4 长班制提醒规则**  
第一次开启长班制时显示完整提示：  
```
长班制已开启。

从今天起，炉城的钟声会比人更晚停下。
所有普通工作建筑产出提高，但患病、疲劳与事故风险也将上升。
恐慌会持续增加，信任会缓慢下降。

长班制将持续生效，直到你手动关闭。

```
后续不重复弹大提示。  
长班制开启期间，在 status 与 end_day 前风险清单中显示短提醒：  
```
当前制度：长班制开启中。
影响：普通工作建筑产出提高；患病、事故风险上升；恐慌将继续增加，信任将继续下降。

```
关闭提示：  
```
长班制已关闭。

今日起，普通工作建筑恢复标准工时。
此前积累的患病、疲劳、事故、信任与恐慌后果不会自动消失。

```
   
⸻  
   
**6. 与儿童 / 学徒线的关系**  
儿童保护所、学堂不属于普通生产建筑，不受长班制影响，也不可被加班日指定。  
儿童辅工或学徒若被分配到受工时制度影响的建筑，则一并承受对应额外风险。  
规则：  
```
儿童辅工所在建筑执行加班日时，儿童同样承担额外风险。
医疗学徒所在医疗站 / 医院执行加班日时，医疗学徒同样承担额外风险。
工程学徒所在研究所执行加班日时，工程学徒同样承担额外风险。
长班制开启时，参与普通工作建筑的儿童辅工与学徒同样承受额外风险。
儿童保护所与学堂不吃长班制。
儿童保护所与学堂不可加班日。

```
说明：  
如果玩家已经让孩子进入工作系统，加班制度不会自动替他们保留童年。  
   
⸻  
   
**7. 与医疗 / 研究建筑的关系**  
医疗站、医院、研究所可被加班日指定。  
**医疗站 / 医院加班日**  
医疗站 / 医院执行加班日时：  
```
当日医疗处理能力或治疗推进方向提高。
医护人员患病 / 疲劳 / 事故风险提高。
医疗学徒若在岗，也承受额外风险。

```
说明：  
* 加班日不免费清空医疗压力。  
* 具体治疗加成、恶化修正和事故风险留到医疗 / 平衡表处理。  
* 学徒不能替代成人兜底规则。  
**研究所加班日**  
研究所执行加班日时：  
```
当日研究推进方向提高。
工程师患病 / 疲劳 / 事故风险提高。
工程学徒若在岗，也承受额外风险。

```
说明：  
* 加班日不提供第二研究队列。  
* 具体研究进度加成留到科技 / 平衡表处理。  
* 学徒不能替代成人兜底规则。  
   
⸻  
   
**8. 防漏洞规则**  
**8.1 加班日不能对空建筑使用**  
```
只有正在运行且有人手的可加班工作建筑，才能执行加班日。

```
**8.2 加班日不能持续遗忘**  
```
加班日是当日急令，日结后自动关闭。

```
**8.3 加班日不能被玩家主动取消**  
```
加班日启动后，玩家不能通过撤人、改工时或再次输入标准工时提前结束。

```
**8.4 长班制不能成为免费产出科技**  
```
长班制开启期间每日持续产生社会代价：
- 恐慌上升
- 信任下降
- 患病 / 疲劳 / 事故风险上升

```
**8.5 长班制持续但必须提醒**  
```
长班制持续到玩家手动关闭。
开启期间必须在 status 与 end_day 前显示提醒。

```
**8.6 非目标建筑不吃加班收益**  
```
儿童保护所、学堂、小酒馆、大赌场、墓园、冷藏坑、仓库、住宅等非普通生产 / 工作建筑，不获得加班日或长班制收益。

```
**8.7 删除建筑不得被工时系统引用**  
```
修造工坊已从 V1 正式建筑中删除。
加班日、长班制、岗位分配、学徒岗位和科技前置均不得引用修造工坊。

```
   
⸻  
   
**9. 本节结论**  
工时 / 劳作作息线形成两层压榨：  
```
加班日：
单建筑、24 小时、当日、急救产出。
需要 confirm。
启动后通常不能主动取消。
日结自动关闭。

长班制：
全城制度、10 小时、持续生效。
产出更稳，但每日积累社会与健康代价。
持续到玩家手动关闭。

```
设计意图：  
* 加班日用于短期救急。  
* 长班制用于长期压榨。  
* 两者都不是免费收益。  
* AI 玩家能直观看懂。  
* 人类旁观能立刻理解这座城已经开始把人当资源烧。  
核心原则：  
炉城可以把人压榨到极限，但不能把这件事伪装成免费产能。 每一小时多出来的产出，都应该在信任、恐慌、疾病、事故或死亡风险里留下回声。
