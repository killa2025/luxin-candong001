# A 【炉心残冬·Patch 006C-2 干净封存版】  
主题：誓言路线 归属：006C 誓言与铁腕炉约 状态：干净封存版  
   
⸻  
   
## 1. 路线定位  
誓言路线是“誓言与铁腕炉约”的其中一条中后期路线。  
它不是超自然信仰，也不是宗教神迹。  
它代表的是：  
```
守炉誓言
共同悼亡
共食忍耐
留城承诺
以意义感承接城市崩裂

```
誓言路线的核心问题是：  
```
当食物、炉火、住房和医疗都不足以让居民安心时，
守炉人是否还能用一个共同誓言，让他们继续留在炉城？

```
誓言路线主要用于承接死亡、饥饿、旧城派动摇与信任崩裂。  
它让居民愿意继续留下，不代表资源、医疗、住房或炉火问题被真正解决。  
   
⸻  
   
## 2. 路线互斥  
签署第一条誓言炉律后，铁腕路线永久关闭。  
规则：  
```
签署“守炉誓言”
→ 锁定誓言路线
→ 永久关闭铁腕路线

```
签署前必须二次确认：  
```
你即将选择誓言路线。
签署后，铁腕路线将永久关闭。

confirm sign 守炉誓言

```
禁止：  
```
先签誓言路线，再切换铁腕路线。
同时拥有誓言路线与铁腕路线。
同时解锁终火誓约与最高戒令。
签署誓言路线后再建造巡查所。

```
   
⸻  
   
## 3. 路线结构  
誓言路线共 6 条炉律：  
```
1. 守炉誓言
2. 悼亡钟声
3. 共食誓餐
4. 余烬名册
5. 留城誓约
6. 终火誓约

```
路线结构：  
```
守炉誓言
├─ 悼亡钟声 → 余烬名册
└─ 共食誓餐 → 留城誓约
      └─ 终火誓约

```
终火誓约是誓言路线最终炉律。  
终火誓约不在路线中段提前开放。  
开放条件：  
```
必须已签署誓言路线前五条炉律。

```
即必须完成：  
```
守炉誓言
悼亡钟声
共食誓餐
余烬名册
留城誓约

```
全部完成后，终火誓约才可进入可签署状态。  
说明：  
```
誓言路线总条目不多。
终火誓约作为最终条，应要求玩家完整走完誓言路线，而不是只完成其中一条分支后提前开放。

```
006C-4 只负责处理：  
```
006C 页面开放时间
006C 签署冷却
主动行动冷却
最终条理论最早可签时间
路线推进节奏

```
006C-4 不改变本条结构前置：  
```
终火誓约必须在誓言路线前五条全部签署后才开放。

```
   
⸻  
   
## 4. 路线代价规则  
誓言路线的代价以熟食消耗、叙事绑定和终局标签为主。  
规则：  
```
被动炉律只在签署当日结算一次性效果或代价。
签署后默认不每日持续提高信任。
签署后默认不每日持续降低恐慌。
主动行动释放时，再结算对应效果与代价。

```
代价方向：  
```
誓言可以承接死亡、饥饿、旧城派动摇与信任崩裂。
但它不能真正解决这些根源。
居民会逐渐依赖共同誓言与守炉叙事维持城市。
终局评价会记录城市依靠终火誓约承接信任崩塌。

```
说明：  
```
誓言路线不是每日自动回血的路线。
它的收益与代价应当在签署炉律或释放行动时清楚结算。

```

## 5. 核心建筑：守炉堂  
守炉堂是誓言路线核心建筑。  
守炉堂是占槽建筑。  
解锁来源：  
```
守炉誓言

```
规则：  
```
守炉堂占普通区域槽位。
守炉堂不使用特殊核心槽。
守炉堂需要玩家手动 build。
守炉堂不会在签署守炉誓言后自动生成。
守炉堂不会自动占槽。
守炉堂上限为 1 座。

```
建筑口径：  
```
守炉堂：占 1 槽，上限 1，推荐内环 / 中环。

```
代码化方向：  
```
守炉堂：
allowed_zones = ["inner", "middle"]
recommended_zone = ["inner", "middle"]
slot_cost = 1
max_count = 1

```
签署“守炉誓言”后：  
```
誓言路线被锁定。
铁腕路线永久关闭。
解锁守炉堂建造权限。
不自动建成守炉堂。
不自动占用建筑槽位。

```
守炉堂不做区域范围效果。  
```
守炉堂建成并运行后，作用于全城誓言路线。
不存在“建在内环 / 中环”导致路线范围不同的问题。
守炉堂所在区域只影响普通槽位占用。

```
可用命令方向：  
```
build 守炉堂 <区域>

```
允许区域：  
```
内环
中环

```
说明：  
```
守炉堂是路线核心建筑，不是炉旁行动效果。
守炉堂占槽。
守炉堂需要手动建造。
守炉堂未建成或未运行时，对应主动行动不可用。

```
   
⸻  
   
## 6. 守炉堂运行规则  
守炉堂是非生产型路线核心建筑，不按人手数量计算效率。  
运行条件：  
```
守炉堂已建成。
守炉堂有人值守。
守炉堂处于运行状态。

```
值守条件：  
```
至少 1 名可工作人口值守即可运行。
普通儿童不可作为工作人员。
儿童辅工 / 学徒不能独立运行。
若使用儿童辅工 / 学徒辅助，至少需要成人兜底。

```
核心判断：  
```
是否已建成。
是否有人值守。
是否满足成人兜底规则。
是否处于运行状态。

```
守炉堂未建成、无人值守、停工或未运行时：  
```
守炉宣誓不可用。
敲悼亡钟不可用。
共食誓餐不可用。
留城劝诫不可用。
终火誓约前置检查不满足。
已签署的永久炉律不失效。

```
永久炉律规则：  
```
已签署的永久炉律不会因为守炉堂未建成、停工或无人值守而失效。
守炉堂运行状态只影响主动行动与终火誓约前置检查，不回滚已经签署的炉律。

```
说明：  
```
守炉堂运行状态主要影响主动行动可用性。
守炉堂未运行不撤销已经签署的永久炉律。

```

## 13. 炉律六：终火誓约  
类型：  
```
誓言路线最终炉律

```
终火誓约是终章炉律，不是建筑，不占槽。  
前置：  
```
必须已签署守炉誓言。
必须已签署悼亡钟声。
必须已签署共食誓餐。
必须已签署余烬名册。
必须已签署留城誓约。
必须已建成守炉堂。
守炉堂必须处于运行状态。
必须满足 006C-4 中规定的路线签署冷却与推进节奏。

```
核心效果：  
```
信任降至 0 时，不再直接失败。
因信任归零或过低触发的失败倒计时不再启动。
若签署前已存在对应信任危机倒计时，则终止或转为非失败警告。

```
签署成功后：  
```
终火誓约的信任失败轴改写成为永久炉律效果。
之后即使守炉堂无人值守、未运行或被后续特殊事件明确禁用，也不会取消终火誓约。

```
仍然有效的失败条件：  
```
存活人口 = 0 仍然失败。
炉心崩毁仍然失败。
恐慌 = 100 且未被最高戒令压制，仍然失败。
资源崩溃造成的死亡链仍然有效。

```
  
说明：    
  
终火誓约只改写信任失败轴。  
终火誓约不免疫恐慌、死亡、饥荒、寒冷、炉心崩毁。  
第七霜落终局验收不作为 006C 新增硬失败条件；低综合分、多系统崩坏、低于城市连续性参考线只进入终局状态评级。  
  
status 显示方向：  
```
信任已崩溃，终火誓约正在承接其后果。
恐慌危机仍然有效。

```
边界：  
```
不免疫恐慌。
不免疫死亡。
不免疫饥荒。
不免疫寒冷。
不免疫旧城派全部后果。
不等于无条件胜利。
不占建筑槽位。
不要求新建额外终章建筑。

```
说明：  
```
终火誓约不是让居民重新相信守炉人。
它是让城市在“不再相信”之后，仍然被一个共同誓言捆住。

```

## 14. 终火誓约后的信任运算规则  
终火誓约只改写信任失败轴的“直接失败判定”，不删除信任数值本身。  
终火誓约生效后：  
```
信任仍然继续计算。
信任仍然继续显示。
信任仍然可以继续下降、维持、恢复。
信任仍然可以被其他系统读取。

```
规则：  
```
信任降至 0 时，不立刻触发信任归零失败。
但信任值仍然保留。
信任仍参与事件、行动、旧城派判断、终局评价和后续系统接口。

```
说明：  
```
失败轴改写 ≠ 信任数值作废。
失败轴改写 ≠ 停止信任结算。
失败轴改写 ≠ 从系统中删除信任状态。

```
信任后续仍可用于：  
```
事件触发
路线行动效果判断
旧城派相关判断
居民状态判断
终局评价
结局标签
旁白文案
其他后续 Patch 的系统接口

```
结论：  
```
终火誓约只改变“信任是否因此直接导致游戏失败”。
不改变信任作为全局城市状态继续运算、显示和被读取的事实。

```
   
⸻  
   
## 15. 被动炉律结算规则  
誓言路线被动炉律只在签署时结算一次性效果或代价。  
```
守炉誓言：签署时开启誓言路线，永久关闭铁腕路线，解锁守炉堂建造权限，启用炉旁誓碑。
悼亡钟声：签署时解锁悼亡钟与对应死亡安抚方向。
共食誓餐：签署时解锁共食台与共食誓餐行动。
余烬名册：签署时解锁余烬名册与死亡余波削弱方向。
留城誓约：签署时解锁留城劝诫。
终火誓约：签署时改写信任失败轴，并留下终局标签。

```
签署后：  
```
不每日自动提高信任。
不每日自动降低恐慌。
不每日自动降低旧城派人数。
不自动建造守炉堂。

```

## 16. 主动行动清单  
誓言路线主动行动共 4 个：  
```
守炉宣誓
敲悼亡钟
共食誓餐
留城劝诫

```
通用规则：  
```
必须解锁对应炉律。
需要守炉堂已建成并处于运行状态。
需要满足对应使用条件。
使用后进入主动行动冷却。
具体冷却由 006C-4 统一确定。

```
主动行动释放时结算固定效果，不使用复杂动态倍率。  
主动行动效果方向：  
```
守炉宣誓：固定小幅提高信任，固定小幅降低恐慌。
敲悼亡钟：近期存在死亡事件时可用，固定降低死亡余波恐慌。
共食誓餐：消耗熟食，固定降低恐慌，固定小幅提高信任。
留城劝诫：旧城派危机激活且旧城派人数 > 0 时可用，固定降低旧城派人数。

```
说明：  
```
誓言路线的主动行动不根据当前信任、恐慌、死亡数量进行复杂倍率计算。
具体固定数值由后续数值表统一确定。

```
   
⸻  
   
## 17. 主动行动边界  
主动行动不会因为守炉誓言已签署而自动执行。  
主动行动仍需满足：  
```
誓言路线已选择。
对应行动已由炉律解锁。
守炉堂已建成。
守炉堂处于运行状态。
行动不在冷却中。
满足行动目标条件。

```
守炉誓言已签署 ≠ 守炉堂已建成。 守炉堂已建成 ≠ 主动行动自动执行。 守炉堂未运行时，誓言主动行动不可用。  
玩家仍需手动使用主动行动。
