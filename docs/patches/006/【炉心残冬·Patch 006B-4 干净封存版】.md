## 一、合并后的干净版正文  
**1. 定位**  
006B-4 处理炉城的人口生存科技。  
它服务以下系统：  
```
狩猎
食堂加工
温室
医疗站
医院
医疗建筑保温
住宅保温
住宅升级
人口抗寒
病患处理
伤残安置建筑保温

```
006B-4 不处理：  
```
社会配给炉律
粗粮汤 / 白米稀粥 / 应急口粮
医疗处置炉律
分级救治
额外医疗配给
儿童保护所 / 学堂路线
资源采集
供暖过载
精神 / 秩序
后期令
药品资源
复杂农业链

```
说明：  
```
006A 负责“怎么分配、怎么牺牲、怎么救治”。
006B-4 负责“工具、建筑与工程能力是否跟得上”。

```

**3. 基础建筑边界**  
以下建筑不由 006B-4 解锁：  
```
基础住宅
食堂
狩猎小屋
医疗站

```
规则：  
```
基础住宅开局可建。
食堂是基础公共建筑，按已有建筑规则建造。
狩猎小屋是基础食物建筑，按已有天气规则运行。
医疗站由 006A 的基础医疗法解锁，不由工程科技解锁。

```
006B-4 可解锁或强化：  
```
温室
医院
改良住宅标准
狩猎效率
食堂效率
医疗效率
医疗建筑保温
住房保温
温室改良

```
说明：  
这条是防死局边界。 不能让玩家“想吃饭要科技，想科技要活过前几天，想活过前几天又要吃饭”。

#### 7.4 温室改良  
  
科技名：温室改良  
Tier：T4  
分类：食物 / 温室升级  
前置：温室栽培  
生效方式：unlock_upgrade  
  
效果方向：  
  
解锁“温室 → 改良温室”的逐座升级权限。  
  
规则：  
  
- 温室改良不是全局 passive。  
- 研究完成后，不会自动强化所有温室。  
- 玩家需要逐座执行 upgrade 温室。  
- 每座温室升级需要材料。  
- 温室升级耗材当前待确认。  
- 升级完成后，该座温室获得更高产出与更好抗寒能力。  
  
不改变：  
  
- 温室数量上限。  
- 温室人手需求。  
- 温室低温停产规则。  
- 第七霜落的终局压力。  
- 无人 / 停工 / 有效温度不足时不能生产的规则。  
  
限制：  
  
- 不提高温室数量上限。  
- 不取消温室低温停产阈值。  
- 不让温室在无人、停工或有效温度不足时继续生产。  
- 不自动升级所有温室。  
  
说明：  
  
温室栽培负责解锁温室建筑。  
温室改良负责解锁温室逐座升级。  
二者不是同一个效果。

```
Tier：T2

生效方式：passive

```

```
医院
```

```
养护所不因此提高治疗速度。
```

```
医疗容量
治疗速度
伤残规则
医疗处置炉律
分级救治规则
额外医疗配给规则

```

```
但医疗容量、治疗制度、分级救治资格仍由医疗建筑与 006A 医疗线处理。
```

#### 9.2 改良住宅标准  
  
科技名：改良住宅标准  
Tier：T2  
分类：住房 / 住宅升级  
前置：住房保温 I  
生效方式：unlock_upgrade  
  
效果方向：  
  
解锁“基础住宅 → 改良住宅”的逐座升级权限。  
  
住宅容量：  
  
- 基础住宅：10 人。  
- 改良住宅：15 人。  
  
规则：  
  
- 研究完成后不会自动升级所有住宅。  
- 玩家仍需手动执行 upgrade 命令。  
- 住宅升级需要材料。  
- 住宅升级耗材当前待确认。  
- 住宅升级不改变占槽。  
- 基础住宅与改良住宅均占 1 槽。  
- 住宅仍不可使用 heat。  
  
不改变：  
  
- 无住所人口当前状态。  
- 住宅自动分配规则。  
- 住宅是否可使用 heat。  
- 住宅占槽。  
- 住房保温科技效果。  
  
说明：  
  
改良住宅标准是容量升级科技。  
它不等于住房保温。  
住房保温 I 负责降低住宅低温风险 / 提高居住安全。  
改良住宅标准负责提高单座住宅容量。  
  
#### 9.3 高级住宅标准  
  
科技名：高级住宅标准  
Tier：T3  
分类：住房 / 住宅升级  
前置：改良住宅标准  
生效方式：unlock_upgrade  
  
效果方向：  
  
解锁“改良住宅 → 高级住宅”的逐座升级权限。  
  
住宅容量：  
  
- 改良住宅：15 人。  
- 高级住宅：20 人。  
  
规则：  
  
- 高级住宅标准正式加入 006B-4。  
- 高级住宅标准只解锁容量升级。  
- 研究完成后不会自动升级所有住宅。  
- 玩家仍需手动执行 upgrade 命令。  
- 住宅升级需要材料。  
- 改良住宅 → 高级住宅升级耗材当前待确认。  
- 高级住宅仍占 1 槽。  
- 高级住宅不可使用 heat。  
- 高级住宅标准不需要修造工坊。  
  
不提供：  
  
- 住房保温。  
- 住宅 heat。  
- 住宅低温风险降低。  
- 自动升级所有住宅。  
- 工坊前置。  
  
说明：  
  
高级住宅标准管容量。
