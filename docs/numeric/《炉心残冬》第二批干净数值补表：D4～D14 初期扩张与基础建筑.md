## 三、D4～D14 阶段定位  
**3.1 阶段范围**  
```
D4～D14：初期扩张与基础建筑接线阶段

```
这一阶段不是科技总表阶段，也不是旧城派阶段。 它主要检验玩家是否能把前 3 天的临时应急转换成基础城市结构。  
**3.2 阶段压力来源**  
D4～D14 的主要压力来源：  
```
1. D4 起临时救护归零，正式医疗站必须接上。
2. 开局无住所人口仍会制造住房与寒冷压力。
3. 开局食物储备开始下降，狩猎小屋 + 食堂必须接上。
4. 第一座狩猎小屋建成后，第 2 猎区自动开放，但仍需要玩家再建第 2 座狩猎小屋才能利用。
5. 木材 / 钢铁必须在住房、食堂、狩猎、医疗站、研究所、儿童保护所之间取舍。
6. 第一批死亡可能出现，墓园 / 冷藏坑开始有意义。
7. 儿童保护压力开始显性化。
8. 第 6 天早期求生者会带来人口压力接口。
9. 地表资源仍能支撑，但不能让玩家误以为中期不用转生产建筑。

```
本批目标：  
```
D4～D14 有明确可走通路径。
玩家不能无视医疗、住房、食物、死亡处理和儿童保护。
玩家不能把第 2 猎区误读成“开局已有两座狩猎小屋上限”。

```

若儿童没有住房，即使被儿童保护所照看，仍按住房系统结算其无住所与寒冷暴露风险。
