# PENDING 登记

本文件是跨模块 PENDING 的施工导航，不替代各权威源文件中的完整待定清单。即使某项未抄录到本页，只要权威源标记为 PENDING、TODO、待确认、待复核或后置，就不得进入正式运行配置或被代码窗自行实现。

权威待定清单至少包括：

- `docs/control/《炉心残冬》全局修正总控正文【代码窗前置必读】.md` 第四节；
- `docs/control/《炉心残冬》干净总收口：七批数值补表代码窗  测试窗交付清单.md` 第十七节；
- `docs/control/《炉心残冬》六轮文案资产表全局总收口  最终终审.md` 第十一、十二节；
- `docs/handoff/《炉心残冬》010 代码骨架输入清单 v0.md` 第十节。
