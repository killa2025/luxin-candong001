
**5. 工程学徒最终岗位**  
工程学徒最终岗位为：  
```
研究所

```
工程学徒不可进入：  
```
修造工坊
所有工程相关岗位
外环高危岗位
未明确标记的工程建筑

```
医疗学徒最终岗位为：  
```
医疗站
医院

```
学徒岗位使用固定名单，不使用“医疗相关 / 工程相关”开放标签。  
   
⸻  
   
**6. heat 与应急加热装置**  
V1 中：  
```
heat 开局可用。

```
前提：  
```
目标建筑支持 heat。
目标建筑满足可加热条件。
玩家支付对应成本。

```
“应急加热装置”不是 heat 解锁科技，而是 heat 强化科技。  
研究所开局可建，但默认不可 heat。  
研究所若后续要可 heat，必须在建筑表单独写：  
```
can_heat: true

```
否则默认不可 heat。  
   
⸻  
   
**7. 守炉堂 / 巡查所**  
守炉堂 / 巡查所为 006C 特殊核心建筑。  
规则：  
```
不占普通建筑槽位。
不需要玩家手动 build。
路线入口签署后自动生效。
必须显示在面板上。
至少 1 名可工作人口值守即可运行。
不可 heat。
不参与普通建筑低温停工。

```
自动生效不等于主动行动自动执行。 主动行动仍需玩家手动使用，并受冷却限制。  
   
⸻

```
额外医疗配给只消耗熟食，不新增医疗物资字段。
```

3 / 20 / 60 暂不进入最终实现。

禁止：

**13. 分级救治**  
分级救治签署时只解锁行动，不直接改变信任 / 恐慌。  
分级救治必须指定：  
```
医疗站
医院

```
不可指定：  
```
养护所
儿童保护所
学堂
研究所
食堂
小酒馆
大赌场
墓园
冷藏坑
仓库
住宅
其他非医疗治疗建筑

```
分级救治不是全城医疗总开关。  
分级救治造成的死亡、伤残、社会影响由死亡系统、伤残系统、社会情绪系统、事件系统统一结算。  
   
⸻  
   
**14. 医疗建筑保温**  
医疗建筑保温作用于：  
```
医疗站
医院
养护所

```
养护所只获得保温与护理风险降低收益。  
养护所不会因此获得：  
```
治疗容量
治疗速度
分级救治资格
急救治疗能力
医疗站 / 医院功能

```
   
⸻  
   
**15. 终局与失败**  
硬失败只保留：  
```
人口归零
信任归零且未被终火誓约承接
恐慌 100 且未被最高戒令压制
炉心崩毁

```
以下不是系统硬失败：  
```
低综合分
多系统崩坏
低于城市连续性参考线
崩坏幸存
残火未灭
玩家主动结束

```
崩坏幸存不是失败。 残火未灭不是失败。 玩家主动结束不是系统硬失败。  
   
⸻  
   
**16. 信任 / 恐慌失败文案**  
信任失败玩家可见结果：  
```
执政官被流放

```
恐慌失败玩家可见结果：  
```
执政官被驱逐

```
信任 / 恐慌失败是玩家失败，不等于炉城立即灭亡。  
通用恐慌失败文案不得混入：  
```
南方传言
旧城派
旧城派出走

```
除非明确触发旧城派专属标签 / 009-C 专属文案。
